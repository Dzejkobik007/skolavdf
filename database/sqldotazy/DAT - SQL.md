# DAT - SQL

## Vytvořte dva SQL soubory pro jednotabulkovou databázi:
```
A) ddl.sql
Create vaší demo tabulky.
Vaše tabulka bude mít sloupec s číslem, textem a datumem. 

B) dml.sql
V tomto demonstrujte SQL DML tak, aby jste ukázali:
    vložení nového záznamu
    úpravu záznamu
    smazání záznamu
    výběr s projekcí
    výběr s projekcí a novým dopočítaným sloupcem
    výběr se selekcí (užijte porovnání i logické spojky)
    výběr s řazením
    výběr s limitem
    práci s textem
    práci s datumem

Každý dotaz bude mít před sebou popisující komentář.

Odevzdejte jako odkaz do vašeho repo na githubu.
```
